Project Overview

This is an API for the Computer Networking Assignment of group MMT class L04.

Technology Stack

Flask: Flask is a lightweight WSGI web application framework in Python. It’s designed to be simple and flexible, making it ideal for small projects and RESTful APIs.

MongoDB: MongoDB is a document-based NoSQL database known for its scalability and flexibility. It stores data in JSON-like documents, providing rich, dynamic, and fast data management.

API Documentation

Peer API

POST /api/peer/add: Adds a new peer. Required data: peer_name, password, status, ip_address, port.

GET /api/peer/<peer_id>: Retrieves information about a peer using peer_id.

PUT /api/peer/<peer_id>/status: Updates the status of a peer. Required data: status.

DELETE /api/peer/<peer_id>: Deletes a peer using peer_id.

GET /api/peer/: Retrieves a list of all peers.

POST /api/peer/<peer_id>/attach: Attaches a file to a peer. Required data: file_path.

GET /api/peer/<peer_id>/content: Retrieves the attached file content of a peer.

Torrent API

POST /api/torrent/add: Adds a new torrent. Required data: torrent_name, peer_ip, peer_port, text_content.

GET /api/torrent/content: Retrieves the content of all torrents.

GET /api/torrent/delete: Deletes all torrents.
